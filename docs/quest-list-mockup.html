Option A: Single Row, Always Visible

All meta inline. Wraps naturally if window is narrow. Simplest approach — everything visible at a glance, but rows get wide with more meta fields.

  • Today 9am Walk the dog Nature, Animal Handling Easy Every 1 day
  • Mar 14 Do laundry Cleaning Moderate Every 3 days
  • Today 2pm Practice guitar Crafting Challenging Every 2 days
  • File taxes Bureaucracy Epic One-off

Option B: Two-Line Rows

Title + primary meta on line 1, secondary meta (skills, attributes) on line 2. Keeps rows from getting too wide. More vertical space but cleaner scan. Scales well when more meta is added in 2F.

  • Today 9am
    Walk the dog Easy Every 1 day
    Nature, Animal Handling
  • Mar 14
    Do laundry Moderate Every 3 days
    Cleaning
  • Today 2pm
    Practice guitar Challenging Every 2 days
    Crafting
  • File taxes Epic One-off
    Bureaucracy

Option C: Expandable Detail

Compact single line by default, click ▸ to expand secondary meta underneath. Cleanest scan, most compact, but requires interaction to see full info. Best if you mostly care about title + due state and only occasionally need details.

  • ▸ Today 9am Walk the dog Easy Every 1 day
    Nature, Animal Handling
  • ▸ Mar 14 Do laundry Moderate Every 3 days
    Cleaning
  • ▸ Today 2pm Practice guitar Challenging Every 2 days
    Crafting
  • ▾ File taxes Epic One-off
    Bureaucracy (showing expanded state as example)
